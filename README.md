## Installation
- Windows
  - Clone the repository (from windows branch) and unzip it
  - Run `.\build.ps1` from within the repository
- Linux
  - Clone the repository (from main branch)
  - SDL2 and SDL2_ttf are required dependencies
  - run `make`
